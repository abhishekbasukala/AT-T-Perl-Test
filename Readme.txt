# Steps to run the program 
Note: I have used xampp for Apache

- If you are using xampp, copy the application in xampp/htdocs.
- Open the pl files and make sure that path to the perl.exe file is correct.
- The database file is db_appointment.sqlite
- Use the url:'http://localhost/appointment-application/' to use the application.
